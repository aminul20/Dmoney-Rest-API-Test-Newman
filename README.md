# Dmoney-Rest-API-Newman

## How to run this project
- Clone this project
- Open with any code editor / Command Shell
- Give this following command ``` npm i ```
- Then ``` node .\report.js ```

## Technology used:
- Postman
- Newman
- Visual Studio Code
